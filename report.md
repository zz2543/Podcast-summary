# 播客总结系统报告

> 本文件从实现阶段开始持续更新，是面向评审者的精炼项目报告。

## 项目概览

本项目实现一个单用户、本地回环访问的播客总结系统，支持本地音频、直链音频与 YouTube 链接输入，并输出结构化摘要、章节、引用、实体列表和可选音频摘要。

## 用户与场景

目标用户需要快速筛选新播客，并在值得深入学习的单集上获得可跳转引用、章节要点和结构化导出。

## 技术选型摘要（来自 research.md）

- ASR：默认豆包语音识别大模型（火山引擎）。
- LLM：默认 DeepSeek OpenAI-compatible endpoint。
- TTS：默认豆包语音合成（火山引擎）。
- 存储：SQLite + `data/<episode_id>/` 文件目录。
- 前端：Vite + React + TypeScript。
